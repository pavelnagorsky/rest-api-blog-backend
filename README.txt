REST-api blog demo 2021 (backend) \n
\n
Функциональность: \n
\n
	Регистрация, авторизация. Отправка, создание, изменение и удаление постов.  \n
	Для операций добавления, редактирования и удаления постов настроен Socket.io \n
	База данных - mongodb & mongoose, сервер реализован с помощью node js & express. \n
	Авторизация - JWT token & bcryptjs \n
	Для хранения и парсинга файлов - multer.js \n
