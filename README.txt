REST-api blog demo 2022 (backend) 

Функциональность: 
	Регистрация, авторизация. Отправка, создание, изменение и удаление постов.  
	Для операций добавления, редактирования и удаления постов настроен Socket.io 
	База данных - mongodb & mongoose, сервер реализован с помощью node js & express. 
	Авторизация - JWT token & bcryptjs 
	Для хранения и парсинга файлов - multer.js
